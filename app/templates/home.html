Welcome to Osmosis!

What is Osmosis?

Osmosis is an alternative score server for the PC rhythm game osu!. Its main goal is to provide an experience that rewards accuracy, as opposed to the official game and server which rewards combo way more.

It is a project that I, Tina, develop and manage on my free time.
The main idea is to see how it would be to play osu! if the only thing that matters were accuracy. Combo scoring is something that other rhythm games players often criticize osu! for. With this project, I'm trying to see if removing the combo factor effectively makes osu! a more pleasant experience for me.
As things went on, I also added or removed other features to the server and client to change osu! a bit in ways I like.

For more information, please visit the About page!
You can find a list of all server and client features there, and details on the server rules!

If you need more information, if you want to involve yourself into ranking or making new charts, or if you need help setting up the client, please join our Discord server.

Plans for the future

  • Add target/pacemaker feature.
  • Group charts of the same song together on the website, even when they are from different beatmaps sets.
  • Authentication without an osu! account.
  • Display server leaderboards in the client.
