Accounts

You do not need to make a seperate account on Project osmosis. Make sure you log into osu! in-game, when submitting scores, your profile username and profile details will be uploaded to build your Osmosis profile.

You may be concerned about what information is stored about you. A few information from your osu! profile is copied on Osmosis to be displayed on the website, such as the date you joined osu!. The server runs the master branch that can be found on GitHub, the player information is described here in the Player model.

Script (RECOMMENDED WAY)

This will download the source code of the game, build it on your computer, then run it.

  1. Download and install Microsoft's .NET Core 3.1 SDK.
  2. Download the latest version of the script for Windows or for Mac OS/Linux.
  3. Put it where you wish to install the game, for example D:\Games\osumosis or C:\Users\*you*\Games\osumosis.
    This is NOT the directory for your local files, such as your map, skins, etc.
  4. Then execute the script. On Windows, you might need to do *Shift+Right Click* to get the option "Run with Powershell".
    On Windows, you might be asked if you trust the script, well, do you?
    The first time you run it, or on every new update, it might take some time! This is normal, you are downloading the files needed to build the game on your computer.
    If the game does not open and the Powershell window closes itself, you encountered an error! Please see the troubleshooting guide below.

    5. Troubleshooting problems with the game on Windows in a few steps:

    1. Make sure osmosis_pusher.exe is not blocked by either your antivirus or Windows. Check its properties.
    2. Open a Powershell window in the script folder. (*Shift+Right Click* in the folder)
    3. If this is the first time you run a Powershell script by hand, you might need to run Set-ExecutionPolicy -ExecutionPolicy Unrestricted -Scope CurrentUser first.
    4. Type .\osmosis_play.ps1, and read the output.

Manual installing (any OS)

  1. Download and install Microsoft's .NET Core 3.1 SDK
  2. Download or clone the source code.
  3. Download the score pusher and put it somewhere in your PATH.

    It has to be named exactly osmosis_pusher or osmosis_pusher.exe!

  4. Open a command prompt in the repository folder, on Windows you can do *Shift+Right Click* and select "Open a Terminal here" or "Open PowerShell here".
  5. dotnet run --project osu.Desktop -c Release to run it directly without making production-ready compiled binaries
