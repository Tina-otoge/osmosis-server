Manual installing

  1. Download Tina's osu! fork
  2. Download the osumosis score pusher and put it somewhere in your PATH.

    It has to be named exactly osmosis_pusher or osmosis_pusher.exe!

  3. Download and install Microsoft's .NET Core 3.0 SDK
  4. Compile Tina's osu! or run it from source:
    • Open a command prompt in the repository folder, on Windows you can do Shift+Right Click and select "Open a Terminal here" or "Open PowerShell here".
    • dotnet run --project osu.Desktop -c Release to run it directly without making production-ready compiled binaries
  5. You are ready to go! When you'll complete songs while playing osu!, your scores will automatically be pushed to the server.

Pre-compiled builds

There are no public builds, however you can try asking me over on osmosis' Discord server.
