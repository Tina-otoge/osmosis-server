Manual installing

  1. Download Tina's osu! fork
  2. Download the osmosis score pusher and put it somewhere in your PATH.

    It has to be named exactly osmosis_pusher or osmosis_pusher.exe!

  3. Download and install Microsoft's .NET Core 3.0 SDK
  4. Compile Tina's osu! or run it from source:
    • Open a command prompt in the repository folder, on Windows you can do Shift+Right Click and select "Open a Terminal here" or "Open PowerShell here".
    • dotnet run --project osu.Desktop -c Release to run it directly without making production-ready compiled binaries
  5. You are ready to go! When you'll complete songs while playing osu!, your scores will automatically be pushed to the server.

Script

Download the latest version of the script, put it somewhere you like, D:\Games\osmosis might be a good place to start. Then execute the script. It should take care of everything.

Your antivirus might be blocking the game, please look out for that!

To troubleshot any problem with the script, go to the script folder and open a Powershell window

If this is the first time you run a PowerShell script by hand, you might need to run Set-ExecutionPolicy -ExecutionPolicy Undefined first.

Type .\osmosis_play.ps1, and read the output.
