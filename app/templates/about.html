About Osmosis

Server rules

  1. Faking scores in any way is prohibited.
    The open source nature of the client and server make it easy to cheat your scores. Don't.
  2. Harassment in any form is prohibited.
    This includes: impersonating other people and "jokingly" insulting a group or category of persons.
  3. NSFW content is prohibited.

The server

Osmosis works differently than the official osu! server. Core differences are listed here. The server is open source.

  • Every score is saved, even for unranked/unsubmitted charts. Even if using unrankable mods such as Relax.
  • Every chart has a leaderboard. You are free to compete for #1 on unranked charts.
  • A points system called osmos is used to rank you globally against other players. The player with the most osmos is the #1 player.
  • Ranked charts's difficulty rating is set manually by humans instead of using osu!'s SR, this new rating is called SSR.
  • SSR goes from 0.5 to 10. It increases by steps of 0.5. Charts rated 5 and more can use 0.25 steps.
    Examples: 1.5⭐, 3.5⭐, 5⭐, 6.25⭐, 7.75⭐, 8.5⭐, 9⭐.
  • Your best play on a ranked chart gives you osmos based on your accuracy and the SSR of the chart.

The client

The client is called osumosis. It is based on the official upcoming osu! client, code-named osu! lazer. The client is open source.

Instructions on how to setup the client and play on the server can be found on the How To page.

Some features were added or removed from the official client:

  • A mod called AR can be used to set the difficulty setting Approach Rate to any value you want. This mod is rankable.
    Using the DA (Difficulty Adjust) mod which was added later on into lazer to change the Approach Rate submits your score as if you used the AR mod.
  • Passive HP drain was removed. You do not lose HP for doing absolutely nothing.
  • A "display FAST/SLOW" setting was added. It helps you timing your hits by showing an extra text above the judgement.
    See this tweet for examples.

Ranked charts

  • Charts ranked on the official osu! server are not necessarly ranked on Osmosis.
  • Charts are ranked on Osmosis by a small team that reviews charts and give them an SSR.
  • Charts are ranked per difficulty, this means ranking a chart does not mean that all the charts in the same beatmap set are necessarly ranked too.
  • If an artist did not express its agreement to have its songs used on osu!, any chart using its songs is unrankable on Osmosis.
  • If a song is licensed in a way that protects its usage, any chart using this song is unrankable on Osmosis.
    This affects but is not limited to songs from the following sources: anime, most video games, most albums.

If you need more information, if you want to involve yourself into ranking or making new charts, or if you need help setting up the client, please join our Discord server.
